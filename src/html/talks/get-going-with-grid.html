What is CSS grid?

  • 2d layout for CSS
  • Before this hacks, <table> / float: left; / display: inline-block;
  • Flexbox only got us so far
  • Items can be placed in rows and columns at the same time

Basic grid

  • Define the grid container display: grid;
  • Create columns grid-template-columns
  • Create rows grid-template-rows
  • Add spacing between items gap – grid-gap

Smarter grid

  • Repeat columns or rows with repeat() function
  • Scaleable items with minmax() function
