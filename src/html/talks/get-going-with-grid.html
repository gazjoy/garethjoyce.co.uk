What is CSS grid?

  • Not a grid system
  • Has nothing to do with Bootstrap
  • Not another layout hack <table> / float: left; / display: inline-block;
{% include 'shared/slide/footer.html' %}

What about flexbox?

{% include 'shared/slide/footer.html' %}

Flexbox only got us so far

  • Flexbox is column or row
  • CSS grid is a 2D layout
  • Items can be placed in rows and columns at the same time
  • Flexbox 💗 Grid
{% include 'shared/slide/footer.html' %}

Demo time ⌚

{% include 'shared/slide/footer.html' %}

Basic grid

  • Define the grid container display: grid;
  • Create columns grid-template-columns
  • Shiny new fr unit
  • Create rows grid-template-rows
  • Add spacing between items gap – grid-gap
{% include 'shared/slide/footer.html' %}

Smarter grid

  • Repeat columns or rows with repeat() function
  • Scaleable items with minmax() function
{% include 'shared/slide/footer.html' %}

Spanning columns

  • Grid lines are numbered and reverse numbered grid-column: 1 / -1;
  • Negate need for col-xs-6 col-sm-8 col-md-12
  • Grid lines can also be named
{% include 'shared/slide/footer.html' %}

Spanning columns and rows

  • Grid lines are numbered the same way on the Y axis, grid-row
  • Items without defined placement get auto placed by the browser
{% include 'shared/slide/footer.html' %}

Now we're cooking 🍳

{% include 'shared/slide/footer.html' %}

Using grid-template-areas

  • Draw a map of your layout
  • Use semantic names
  • Redefine grid at different screen sizes
  • Mapping items is already done
{% include 'shared/slide/footer.html' %}

Using @media

  • Grid coupled with @media is powerful
  • Design mobile first, you may not need a grid
{% include 'shared/slide/footer.html' %}

Implicit grid

  • When you have more items than what is explicitly defined
  • Can be controlled with auto placement
  • grid-auto-rows / grid-auto-flow
{% include 'shared/slide/footer.html' %}

Fit as many items as possible

  • No media queries!
  • Use auto-fill / auto-fit with repeat
{% include 'shared/slide/footer.html' %}

Using @supports

  • No polyfill for grid
  • Use progressive enhancement
  • Modernizr for CSS
  • Can use grid without feature queries
  • Always check your analytics for browser usage
{% include 'shared/slide/footer.html' %}

Always remember ⚠️

Websites do not need to look the same on every browser!
